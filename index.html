How It Works: The Core Logic

The magic is in `script.js`. It listens for clicks on navigation links and shows or hides content sections accordingly.

1. HTML Structure

The site is built around a simple but powerful structure. The `

` tag holds multiple `
` elements. Each section acts as a "page".

<main id="main-content">
    <section id="home" class="content-section active-section">
        <!-- Home page content -->
    </section>
    <section id="how-it-works" class="content-section">
        <!-- This section's content -->
    </section>
    <!-- Add more sections here -->
</main>

Notice that only the "home" section has the `active-section` class initially. The others are hidden by default via CSS (`.content-section { display: none; }`).

2. JavaScript Navigation

When you click a navigation link (e.g., "How It Works"), the JavaScript performs these steps:

  1. Prevents Default Action: It stops the link from trying to jump to an anchor (`e.preventDefault()`).
  2. Gets Target ID: It reads the `data-section` attribute from the link you clicked (e.g., `data-section="how-it-works"`).
  3. Switches Classes: It loops through all `content-section` elements. It removes `active-section` from the currently visible section and adds it to the target section.
  4. Updates Nav Highlight: It updates the `.active` class on the navigation links to highlight the current page.

Here is the key function from `script.js` that handles this:

const showSection = (targetId) => {
    contentSections.forEach(section => {
        if (section.id === targetId) {
            section.classList.remove('hidden');
            section.classList.add('active-section');
            window.scrollTo({ top: 0, behavior: 'smooth' });
        } else {
            section.classList.add('hidden');
            section.classList.remove('active-section');
        }
    });
    setActiveLink(targetId);
};

Customizing Your Site

Follow these steps to make this template your own.

1. Change the Color Scheme

Open `style.css`. At the top, you'll find CSS variables in the `:root` block. Simply change these color codes to match your brand.

:root {
    --sunny-yellow: #FFDA63; /* Change this */
    --sky-blue: #A0D2DB;     /* Change this */
    --coral: #FF7F50;        /* Change this */
    /* ... and so on */
}

2. Add a New Section (Page)

Let's say you want to add an "About" page.

  1. In `index.html` (Header): Add a new link to the navigation list. 
    <li><a href="#about" class="nav-link" data-section="about">About</a></li>
  2. In `index.html` (Main Content): Add the new section inside the `
    ` tag. Copy the structure of another section. 
    <section id="about" class="content-section">
    <div class="container section-container">
        <h2>About Us</h2>
        <p>Your new page content goes here!</p>
    </div>
</section>

That's it! The JavaScript will automatically detect the new section and link, and everything will work without any changes to `script.js`.

3. Edit Interactive Content

To change the text generated by the "Generate Text" button on the Components page, find the `compliments` array in `script.js` and edit the strings.

const compliments = [
    "Your new string one.",
    "Your new string two.",
    "And so on..."
];
